## JS vs TS Which is better ?

### Javascript

```
- Linh hoạt
- Không cần phải định nghĩa kiểu dữ liệu hoặc tìm hiểu kiểu dữ liệu khi sử dụng libs
- Code nhanh hơn khi làm một mình
- Cần phải hiểu rõ về Javascript để tránh những lỗi lặt vặt
=> Đi nhanh khi làm việc mình và lúc ban đầu, càng về sau khi dự án lớn dần thì xảy ra vấn đề không mong muốn
```

### Typescript

```
- Gặp khó khăn với type system vì bị lỗi liên quan tới kiểu dữ liệu 
- Được cái nhắc code 
- Sau khi quen dần với TS thì code sẽ tự tin hơn, đỡ lỗi lặt vặt
- Ban đầu sẽ mất chút thời gian để tìm hiểu kiểu dữ liệu để khai báo 
- Khi làm việc với team thì sẽ rất tiện lợi từ việc auto complete
=> Đi hơi chậm lúc ban đầu, nhưng lúc sau đi sẽ nhanh và an toàn hơn
```

## Can I switch from JS to TS?

```
- Chuyển sang Typescript được nhưng hãy cân nhắc sử dụng any 
- Dần dần sẽ refactor any thành type tương ứng cho đến cover hết thì thôi
```
