## Truthy and falsy

### What is truthy / falsy

```
- Falsy values là các giá trị liệt kê trong bảng dưới
    + false
    + 0 
    + -0
    + 0n
    + "", '',  ``
    + null
    + undefined
    + NaN
    + document.all
- Ngược lại, tất cả là truthy (kể cả Object rỗng, mảng rỗng)
```
### Use Case

```
- !! operator or Boolean()
```
